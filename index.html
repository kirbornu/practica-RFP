Справка по ручному редактированию файла маршрутизации

В общем файл должен выглядить так 

      function FindProxyForURL(url, host)
      {
        $Proxy = "PROXY proxy.rfpgroup.ru:8090";

        //PROXY
        if (shExpMatch(host, "example.com") || shExpMatch(host, "*.example.com")) {
              return "PROXY 172.28.0.100:3128";
        }
        ...

        //DIRECT
        if (shExpMatch(url, "*//example.ru/*")) {return "DIRECT";}
        ...

        //constants
        ...

        alert("Return" + $Proxy);

        return $Proxy;
      }

Крайне необходимо расставить маркеры //PROXY, //DIRECT и //constants, потому что парсер скрипта ориентируется на них для работы с файлом.

Все прокси правила ставятся сразу после //PROXY, все директ правила после //DIRECT, а после //constants стоят константы - их отличие заключается в том, что парсер вообще их не видит, то есть удалить их через редактор невозможно.

Все новые правила будут добавлять в верх списка под соответствующим маркером. При отсутствии маркеров правила будут добавлять после определения $Proxy.
